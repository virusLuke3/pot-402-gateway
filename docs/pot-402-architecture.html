POT-402 Gateway architecture

A safe-mode Portaldot payment gateway demo. It shows the complete HTTP 402 → payment challenge → receipt → unlock loop without broadcasting transactions.

Demo proof

  • HTTP 402 challenge
  • Receipt verification
  • Paid API unlock

Portaldot proof

  • POT token/decimals
  • SS58 42
  • Read-only RPC status

Safety

  • No private keys
  • No tx broadcast
  • Real tx needs approval
