POT-402 Gateway

A Portaldot-native 402 Payment Required gateway for pay-per-call APIs. Builders return a POT payment challenge, verify a receipt, and unlock premium resources.

Safe by default: mock mode does not broadcast. Local-dev mode only broadcasts to ws://127.0.0.1:9944 after you start portaldot_dev --dev --alice; it never targets mainnet.

1. Payment Challenge

Protected endpoints return structured HTTP 402 JSON with Portaldot network details and a suggested `balances.transferKeepAlive` payment.

Click “Call Protected API” to start.

2. Receipt Verification

Mock mode proves the API flow. Local-dev mode submits a real balances.transferKeepAlive extrinsic to your local Portaldot dev node using Alice's public dev account.

3. Unlocked API

After verification, the access token unlocks the paid endpoint and records proof in the gateway ledger.
